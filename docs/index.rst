###########
Aldryn Jobs
###########


Aldryn Jobs is an `Aldryn <http://aldryn.com>`_-compatible jobs application for for `django CMS
<http://django-cms.org>`_.

Aldryn Jobs allows you to publish job openings on your django CMS website. It also allows you
administer the recruitement process, by receiving applications and candidates' CVs via the website.

Aldryn Jobs is `open-source software <https://github.com/aldryn/aldryn-jobs>`_.
